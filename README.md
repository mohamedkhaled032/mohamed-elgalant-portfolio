# Mohamed Khaled - Professional Flutter Portfolio

A responsive single-page portfolio for a Flutter Developer. It includes selected case studies for ArabTesting Platform, Psychological Assessment Tools, Autism Learning Application, Android Build & Release Modernization, BeBeauty App, and Dar Al-Mustafa.

## Included
- `index.html` - complete website entry point
- `assets/Mohamed_Khaled_Flutter_CV.pdf` - latest ATS-oriented CV used by the website Download CV button
- `assets/Mohamed_Khaled_Flutter_CV_ATS_Final.docx` - editable CV copy
- `assets/mohamed-khaled-profile.png` - portfolio image
- `DEPLOY_FROM_CMD.txt` - safe Windows CMD deployment commands

## Publish to GitHub Pages
Use the commands in `DEPLOY_FROM_CMD.txt`. Push to the `main` branch of the existing repository:
`https://github.com/mohamedkhaled032/mohamed-elgalant-portfolio.git`

The published website URL is:
`https://mohamedkhaled032.github.io/mohamed-elgalant-portfolio/`
